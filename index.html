Pengaturan Firestore Rules

Untuk aplikasi ini berfungsi, pastikan Firestore Rules Anda:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /transactions/{transactionId} { allow read, write: if request.auth != null; } } }
Rules ini mengizinkan anonymous users (otomatis dibuat saat app dimuat) untuk membaca dan menulis data ke koleksi 'transactions'.
